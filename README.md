# Annotator Pro

## Setup
Clone this repository and ensure you have installed Python 3.7+ and all dependencies on your system.
```commandline
> git clone https://github.com/aaravdave/Annotator-Pro.git
> cd [FILE DIRECTORY]
> pip install -r requirements.txt
> python3 main.py
```
